Run analyzer from the nested Flutter app directory:

cd booknest-mobile-app-40323-40332/booknest_mobile_frontend
flutter pub get
flutter analyze

Or from repository root, use the canonical scripts:
- bash RUN_ANALYZE.sh        # uses system Flutter
- bash run_ci.sh             # bootstraps Flutter if missing

If your CI cannot change working directory, configure it to execute:
- bash booknest-mobile-app-40323-40332/booknest_mobile_frontend/analyze.sh
